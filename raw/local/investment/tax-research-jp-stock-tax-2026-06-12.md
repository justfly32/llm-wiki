## 1. 일본 세제 개요

### 1.1 과세 체계

일본은 **자진신고 평가납부 방식**을 채택하고 있으며, 국세청(National Tax Agency, NTA)이 관할합니다.

| 구분 | 내용 |
|------|------|
| **과세기간** | 1월 1일 ~ 12월 31일 (역년 단위) |
| **신고 기한** | 매년 2월 16일 ~ 3월 15일 |
| **납부 기한** | 신고 기한과 동일 (분할 납부 가능) |
| **세무 당국** | 국세청 (NTA) |

### 1.2 거주자 구분과 과세 범위

| 거주 구분 | 정의 | 과세 범위 |
|-----------|------|-----------|
| **비거주자 (Non-Resident)** | 일본 체류 1년 미만 | 일본 국내 원천소득만 |
| **비영주 거주자 (Non-Permanent Resident)** | 외국인, 최근 10년 내 일본 체류 5년 이하 | 일본 국내 소득 + 일본으로 송금된 해외 소득 |
| **영주 거주자 (Permanent Resident)** | 최근 10년 내 일본 체류 5년 초과 | 전 세계 소득 |

> ⚠️ **비영주 거주자 주의사항**: 해외 신용카드를 일본에서 사용하는 것도 "송금(remittance)"으로 간주될 수 있습니다.

### 1.3 소득세 누진세율 (2025년 기준)

일본 국민소득세는 7단계 누진세율이 적용됩니다. 여기에 주민세(약 10%)가 별도 부과되어 최고 실효세율은 약 55%에 달합니다.

| 과세소득 구간 (엔) | 국민소득세율 | 주민세 (부+시) | 합계 실효세율 |
|---------------------|-------------|----------------|--------------|
| ~ 195만 | 5% | 10% | ~15% |
| 195만 ~ 330만 | 10% | 10% | ~20% |
| 330만 ~ 695만 | 20% | 10% | ~30% |
| 695만 ~ 900만 | 23% | 10% | ~33% |
| 900만 ~ 1,800만 | 33% | 10% | ~43% |
| 1,800만 ~ 4,000만 | 40% | 10% | ~50% |
| 4,000만 이상 | 45% | 10% | ~55% |

> **재건특별소득税**: 2011년 동일본 대지진 복구를 위해 국민소득세액의 2.1%를 2037년까지 추가 부과

## 2. 양도소득세 (Capital Gains Tax)

### 2.1 상장 주식·증권

| 항목 | 내용 |
|------|------|
| **세율** | **20.315%** (정률) |
| **구성** | 국민소득세 15% + 재건특별소득세 0.315% + 주민세 5% |
| **보유기간 할인** | 없음 (단기·장기 동일 세율) |
| **과세 방식** | 신고분리과세 (근로소득과 합산하지 않음) |

### 2.2 비상장 주식

| 항목 | 내용 |
|------|------|
| **세율** | **20.315%** (상장 주식과 동일) |
| **손익통산** | 상장 주식과 구분 과세 (링펀싱) |

### 2.3 부동산 양도소득세 (참고)

| 보유기간 | 세율 | 구성 |
|----------|------|------|
| **단기 (5년 이하)** | **39.63%** | 국민소득세 30% + 재건세 0.63% + 주민세 9% |
| **장기 (5년 초과)** | **20.315%** | 국민소득세 15% + 재건세 0.315% + 주민세 5% |

> 주택 양도 시 최대 3,000만엔 비과세 특례 적용 가능

### 2.4 양도소득세 계산 방식

```
양도소득금액 = 양도가액 - (취득원가 + 필요경비)
양도소득세 = 양도소득금액 × 20.315%
```

- **필요경비**: 매매 수수료, 이체 수수료 등 직접 비용 포함
- **취득원가**: 선입선출법 또는 이동평균법 선택 가능 (계좌 개설 시 지정)

## 3. 배당소득세 (Dividend Tax)

### 3.1 기본 세율

| 항목 | 내용 |
|------|------|
| **세율** | **20.315%** (양도소득세와 동일) |
| **구성** | 국민소득세 15% + 재건특별소득세 0.315% + 주민세 5% |

### 3.2 과세 방식 선택

일본 거주자는 배당소득에 대해 두 가지 과세 방식 중 선택할 수 있습니다:

| 과세 방식 | 설명 | 세율 |
|-----------|------|------|
| **원천분리과세 (신고불요)** | 배당 지급 시 20.315% 원천징수로 종결 | 20.315% |
| **종합과세 (신고필요)** | 다른 소득과 합산하여 누진세율 적용 | 15% ~ 55% |

> **선택 기준**: 연간 배당소득이 2,000만엔 이하이면 원천분리과세가 유리한 경우가 많습니다. 고소득자의 경우 종합과세 시 세율이 더 높아질 수 있습니다.

### 3.3 배당소득 공제

종합과세 선택 시, 배당소득에 대해 일정 비율의 소득공제가 적용됩니다:

| 배당소득 구간 | 공제율 |
|---------------|--------|
| 1,000만엔 이하 | 10% |
| 1,000만엔 초과 ~ 5,000만엔 이하 | 5% |
| 5,000만엔 초과 | 2.5% |

### 3.4 소액배당 신고불요 제도

- 1회 지급액이 **100만엔 이하**인 배당금은 신고하지 않아도 되는 제도
- 원천징수세(20.315%)로 과세 종결

## 4. NISA (니사) — 소액투자 비과세제도

### 4.1 개요

NISA(Nippon Individual Savings Account)는 일본 정부가 국민의 자산 형성을 지원하기 위해 도입한 **투자 수익 비과세 제도**입니다. 2024년 1월 대폭 개편되어 더욱 강력한 혜택을 제공합니다.

### 4.2 새로운 NISA (2024년 개편) 주요 내용

| 항목 | 내용 |
|------|------|
| **비과세 대상** | 주식/ETF/펀드 매매차익, 배당금, 분배금 등 모든 투자 수익 |
| **연간 투자 한도** | **360만엔** (적립식 120만엔 + 성장투자 240만엔) |
| **평생 비과세 한도** | **1,800만엔** (누적 투자 원금 기준) |
| **비과세 보유 기간** | **무기한** (기존 5~20년 제한 폐지) |
| **비과세 한도 복원** | 보유 자산 매각 시 해당 원금만큼 다음 해 한도 복원 |
| **가입 자격** | 일본 거주 18세 이상 (국적·비자 종류 무관) |
| **계좌 수 제한** | 1인당 1개의 금융기관에서만 개설 가능 |

### 4.3 두 가지 투자 유형

| 구분 | つみたて (적립식) | 성장투자 (자유식) |
|------|-------------------|-------------------|
| **연간 한도** | 120만엔 | 240만엔 |
| **투자 방식** | 매월 정기 적립 | 자유롭게 투자 |
| **투자 가능 상품** | 정부 인증 저위험 펀드 | 개별 주식, ETF, REIT, 액티브 펀드 등 |
| **특징** | 장기 분산투자 적합 | 단기 매매·개별 종목 투자 적합 |

### 4.4 NISA 절세 효과 예시

| 구분 | 일반 계좌 | NISA 계좌 |
|------|-----------|-----------|
| 투자 원금 | 1,000만엔 | 1,000만엔 |
| 수익률 | 10% (100만엔) | 10% (100만엔) |
| 세금 | 약 20.3만엔 | **0엔** |
| 실수령액 | 약 79.7만엔 | **100만엔** |

> 장기 복리 효과를 고려하면 그 차이는 더욱 커집니다.

### 4.5 외국인 유의사항

- **미국 시민권자**: NISA는 IRS에서 인정하지 않으며, 일본 뮤추얼 펀드는 PFIC(외국투자회사)로 분류되어 불리한 세금 처리를 받을 수 있습니다. 개별 주식이나 미국 본거지 ETF만 보유 권장.
- **한국 거주자**: 일본 NISA 가입 자체는 가능하나, 한국 세법상 해외소득으로 신고 의무 발생 가능. 한일조세조약 확인 필요.

### 4.6 기존 NISA vs 신 NISA 비교

| 항목 | 기존 NISA (2023년까지) | 신 NISA (2024년~) |
|------|------------------------|-------------------|
| 연간 한도 | 120만엔 (일반) / 40만엔 (적립) | 360만엔 (적립 120 + 성장 240) |
| 비과세 기간 | 5년 / 20년 | **무기한** |
| 누적 한도 | 600만엔 / 800만엔 | **1,800만엔** |
| 한도 복원 | 없음 | **매각 시 복원** |
| 제도 기간 | 2014~2023 (유한) | **상설화** |

## 5. iDeCo (개인형 확정기여형 연금)

### 5.1 개요

iDeCo(Individual Defined Contribution pension)는 일본의 개인 연금 저축 제도로, **납부금 전액 소득공제**와 **운용 수익 비과세** 혜택을 제공합니다.

### 5.2 2025년 월 납부 한도

| 가입자 유형 | 월 최대 납부액 |
|-------------|---------------|
| **1종 피보험자** (자영업자, 학생 등) | 68,000엔 → **75,000엔** (2027년~) |
| **2종 피보험자** (회사원, 공무원) - 기업연금 없음 | 23,000엔 → **62,000엔** (2027년~) |
| **2종 피보험자** - 기업DC만 가입 | 20,000엔 (55,000 - 기업납부분) |
| **2종 피보험자** - 확정급여형(DB)만 가입 | 20,000엔 (55,000 - DB 상당액) |

### 5.3 세금 혜택

| 단계 | 세금 혜택 |
|------|-----------|
| **납부 시** | 납부금 전액 소득공제 (연금 소득공제) |
| **운용 중** | 운용 수익 100% 비과세 |
| **수령 시** | 일시금: 퇴직소득공제 / 연금: 공제금 공제 |

### 5.4 iDeCo vs NISA 비교

| 항목 | iDeCo | NISA |
|------|-------|------|
| **납부금 소득공제** | ✅ 전액 공제 | ❌ 없음 |
| **운용 수익 비과세** | ✅ | ✅ |
| **인출 자유도** | 60세 이전 제한적 | 언제든 가능 |
| **연간 한도** | 최대 75,000엔/월 | 360만엔/년 |
| **적합 대상** | 장기 은퇴 준비 | 유연한 투자 |

> **전략**: 장기 거주 예정이면 iDeCo를 먼저 극대화하고, 나머지를 NISA로 활용하는 것이 효과적입니다.

## 6. 비거주자 (외국인) 세금

### 6.1 비거주자 과세 범위

| 거주 구분 | 과세 대상 |
|-----------|-----------|
| **비거주자** | 일본 국내 원천소득만 |
| **비영주 거주자** | 국내 소득 + 일본 송금 해외 소득 |
| **영주 거주자** | 전 세계 소득 |

### 6.2 비거주자 주식 관련 세금

| 소득 유형 | 세율 | 비고 |
|-----------|------|------|
| **상장주 양도차익** | **20.315%** (원천분리과세) | 일본 거주자와 동일 |
| **상장주 배당** | **15.315%** (원천징수) | 조세조약에 따라 감면 가능 |
| **비상장주 양도차익** | **20.315%** | |
| **부동산 양도** | **10.21%** (매수인 원천징수) | |

### 6.3 조세조약에 의한 세율 감면

일본은 80개 이상의 국가와 조세조약을 체결하고 있습니다. 주요 조세조약 배당 원천징수세율:

| 국가 | 배당 원천징수세율 |
|------|-------------------|
| **미국** | 10% (일반) / 0% (연금 등) |
| **한국** | 15% (일반) / 5% (지분 10% 이상) |
| **영국** | 10% (일반) / 0% (연금 등) |
| **중국** | 10% |
| **싱가포르** | 15% (일반) / 5% (지분 25% 이상) |

> 조세조약 적용을 원하면 W-8BEN (미국인) 또는 해당 양식을 증권사에 제출해야 합니다.

### 6.4 비거주자 특별 사항

- **특정 口座 (Tokutei Koza)**: 일본 증권사에서 개설하는 특별 계좌를 통해 자동 원천징수 가능
- **신고 의무**: 비거주자의 경우 일반적으로 원천징수로 과세가 종결되나, 일부 경우 별도 신고 필요
- **부동산 매각**: 매수인이 거주자인 경우, 매수인이 매매대금의 10.21%를 원천징수하여 세무서에 납부

## 7. 원천징수 제도

### 7.1 특별 계좌 (Tokutei Koza / 特定口座)

일본 증권사에서 개설할 수 있는 원천징수 전용 계좌로, **세금 계산과 납부가 자동화**됩니다.

| 유형 | 설명 |
|------|------|
| **원천징수 계좌 (源泉徴収口座)** | 매매차익·배당에 대해 20.315% 자동 원천징수. 별도 신고 불필요 |
| **일반 계좌 (一般口座)** | 직접 세금 계산 및 신고 필요. 증권사가 연간 거래명세서 발행 |

> **권장**: 대부분의 투자자는 원천징수 계좌를 사용하면 신고 절차가 간소화됩니다.

### 7.2 원천징수 세율 요약

| 소득 유형 | 거주자 | 비거주자 |
|-----------|--------|----------|
| **상장주 양도차익** | 20.315% | 20.315% |
| **상장주 배당** | 20.315% | 15.315% (조세조약 적용 시 감면) |
| **이자 소득** | 20.315% | 15.315% |
| **부동산 임대소득** | 20.42% | 20.42% |

## 8. 손익통산 및 손실 이월

### 8.1 손익통산 (損益通算)

일본은 **금융투자 상품 간 손익통산**을 인정합니다:

| 통산 가능 항목 | 비고 |
|----------------|------|
| **상장주 양도손실 ↔ 상장주 양도차익** | ✅ 가능 |
| **상장주 양도손실 ↔ 배당소득** | ✅ 가능 |
| **비상장주 ↔ 상장주** | ❌ 불가 (링펀싱) |
| **부동산 ↔ 주식** | ❌ 불가 |

> **예시**: 주식 배당으로 1,000만엔을 벌었으나 매매로 1,000만엔 손실이 발생하면 과세 대상이 0엔이 됩니다.

### 8.2 손실 이월 (繰越控除)

| 항목 | 내용 |
|------|------|
| **이월 기간** | **3년** (손실 발생 이후 3년 내 차익과 상계) |
| **신고 조건** | 반드시 해당 연도에 **확정신고**를 해야 이월 가능 |
| **적용 대상** | 상장주 양도손실, 비상장주 양도손실 (각각 별도) |

> ⚠️ **중요**: 원천징수 계좌를 사용하면 자동으로 손실 이월이 처리되지 않을 수 있습니다. 손실 이월을 활용하려면 확정신고가 필수입니다.

## 9. 조세조약 및 외국납부세액공제

### 9.1 외국납부세액공제

일본 거주자가 해외에서 납부한 세금은 일본 소득세에서 공제받을 수 있습니다:

```
공제한도 = 일본 소득세 × (해외 소득 ÷ 전 세계 총소득)
```

- **초과 공제분**: 3년간 이월 가능
- **공제 방법**: 일본 확정신고 시 외국납부세액공제 신청

### 9.2 한일조세조약 주요 내용

| 소득 유형 | 일본 내 원천징수세율 | 비고 |
|-----------|---------------------|------|
| **배당** | 15% (일반) / 5% (지분 10% 이상) | |
| **이자** | 10% | |
| **로열티** | 10% | |
| **양도차익** | 거주국 과세 원칙 | 부동산 제외 |

## 10. 퇴출세금 (Exit Tax)

### 10.1 적용 대상

일본을 떠나는 장기 거주 외국인이 다음 조건을 충족하면 퇴출세금이 적용됩니다:

| 조건 | 내용 |
|------|------|
| **보유 금융자산** | **1억엔 이상** |
| **체류 기간** | 최근 10년 내 5년 이상 일본 거주 |
| **적용 시점** | 일본 출국 시 |

### 10.2 과세 방식

- **미실현 차익**에 대해 **15.315%** 세율 적용
- 매각하지 않아도 세금 부과 (시가 기준 평가)
- **유예 제도**: 담당 세무대리인 선임 시 최대 5년 (연장 시 10년) 유예 가능
- **면제**: 테이블 1 비자 (일반 취업비자) 소지자는 일반적으로 면제

## 12. 증권거래세

### 12.1 현행

일본은 **1999년에 증권거래세를 완전 폐지**했습니다.

| 시기 | 증권거래세율 |
|------|-------------|
| **1989년 이전** | 0.3% ~ 0.55% |
| **1989~1999년** | 단계적 인하 |
| **1999년 이후** | **0% (폐지)** |

> 일본은 1987년 세제개편을 통해 1989년 양도소득세를 도입하고, 10년에 걸쳐 증권거래세를 단계적으로 폐지했습니다. 이는 거래 위축을 방지하면서 과세 형평성을 높이기 위한 조치였습니다.

## 15. 절세 전략 요약

### 15.1 기본 전략

1. **NISA 극대화**: 연간 360만엔 한도까지 반드시 활용
2. **iDeCo 우선**: 장기 거주 시 납부금 소득공제 혜택이 큼
3. **원천징수 계좌 활용**: 신고 절차 간소화
4. **손실 이월 활용**: 확정신고 필수, 3년 내 차익 상계

### 15.2 고급 전략

1. **손익통산 전략**: 배당 수입이 큰 해에 손실 실현 (tax-loss harvesting)
2. **거주 기간 관리**: 비영주 거주자 기간 동안 해외 소득 송금 최소화
3. **조세조약 활용**: W-8BEN 등 양식 제출로 원천징수세율 감면
4. **퇴출세금 대비**: 1억엔 이상 금융자산 보유 시 출국 전 전략 수립

### 15.3 외국인 투자자 체크리스트

- [ ] 일본 거주 여부 및 거주 기간 확인
- [ ] 조세조약 적용 가능 여부 확인
- [ ] NISA 계좌 개설 (18세 이상 거주자)
- [ ] 원천징수 계좌 vs 일반 계좌 선택
- [ ] W-8BEN (미국인) 또는 해당 양식 제출
- [ ] 해외자산 신고 의무 확인 (5,000만엔 이상)
- [ ] 퇴출세금 적용 여부 확인 (출국 시)
