# Diagnosing Task Insensitivity in Language Agents

> Source: [arXiv AI](https://arxiv.org/abs/2606.26918)  
> Date: 2026-06-26

arXiv:2606.26935v1 Announce Type: new 
Abstract: Chain-of-thought (CoT) reasoning is widely used in language-model agents, but prior work has shown that verbalized CoT is not always faithful and may instead reflect post-hoc reasoning, which means the model already knows the answer before reasoning.
