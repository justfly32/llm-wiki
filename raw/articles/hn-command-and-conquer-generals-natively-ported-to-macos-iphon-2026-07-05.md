No emulation: this is the real 2003 engine compiled for ARM64, rendering DirectX 8 → DXVK → Vulkan → MoltenVK → Metal. Built on EA's GPL v3 source release via fbraz3/GeneralsX (which did the heavy lifting of the macOS/Linux port — this fork adds the iOS/iPadOS port and a set of engine fixes). The original GeneralsX README lives on the upstream-main branch. No game assets are included or distributed. You need your own copy ( Steam , ~$5 on sale).
