# Vlasov-Poisson 시스템에 대한 동형(homeomorphic) 수정 파동 연산자(modified wave operators)

> 📄 원문: [Homeomorphic modified wave operators for the Vlasov-Poisson system](https://arxiv.org/abs/2606.06488v1)
> ✍️ 저자: Léo Bigorgne
> 📅 발행: 2026-06-04
> 🏷️ 카테고리: math.AP

## 요약

우리는 초기 데이터, 산란 상태, 점근적 수렴이 동일한 위상에서 측정되는 함수적 틀 내에서 Vlasov-Poisson 시스템의 소 데이터 해에 대해 수정된 산란(modified scattering)을 증명한다. 또한, 파동 연산자(wave operators)가 초기 데이터 공간과 산란 데이터 공간 사이의 위상동형사상(homeomorphisms)을 정의하며, 더 약한 노름에서 국소 립시츠 연속성(local Lipschitz continuity property)을 가짐을 보인다. 그 결과, 척력(repulsive) 경우 대칭 구형 대 데이터 해는 점근적으로 안정적이다. 증명은 특히 점근적 비선형 흐름에 적합한 동적 좌표계(dynamic coordinates) 시스템의 도입에 의존한다.
